**Update src/server.ts — handleHealthRoute:**

Replace the existing handleHealthRoute function with a version that:
1. Records server start time at module initialization: `const SERVER_START_TIME = Date.now();`
2. Attempts to call the Convex ping function (use getConvexClient() and call the ping query)
3. Returns JSON: `{ "status": "ok", "version": "1.0.0", "uptime": Math.floor((Date.now() - SERVER_START_TIME) / 1000), "convex": "ok" }`
4. On Convex timeout/error, returns status 200 but with `"status": "degraded"` and `"convex": "error"` (the server is up, but a dependency is unhealthy)
5. Uses a 3-second timeout for the Convex ping to avoid blocking health checks

```typescript
const SERVER_START_TIME = Date.now();
const APP_VERSION = process.env.npm_package_version ?? "1.0.0";

async function handleHealthRoute(_req: IncomingMessage, res: ServerResponse): Promise<void> {
  let convexStatus: "ok" | "error" = "ok";

  try {
    const convex = getConvexClient();
    const pingResult = await Promise.race([
      convex.query("ping:ping" as any, {} as any),
      new Promise((_, reject) => setTimeout(() => reject(new Error("timeout")), 3000)),
    ]);
    convexStatus = (pingResult as any)?.ok === true ? "ok" : "error";
  } catch {
    convexStatus = "error";
  }

  const status = convexStatus === "ok" ? "ok" : "degraded";
  const uptimeSeconds = Math.floor((Date.now() - SERVER_START_TIME) / 1000);

  sendJson(res, 200, {
    status,
    version: APP_VERSION,
    uptime: uptimeSeconds,
    convex: convexStatus,
  });
}
```

Note: `npm_package_version` is set automatically by Node.js when running via npm scripts. Add a fallback of "1.0.0".

Check convex/ping.ts to confirm the correct function name (likely `ping:ping`). Adjust the Convex call path accordingly.
