**convex/dashboardStats.ts** — Add two new time-windowed counts to getDashboardStats:
- `messagesToday`: count of chat_messages where `createdAt >= startOfTodayMs`. To compute start-of-today: `const now = Date.now(); const d = new Date(now); d.setHours(0,0,0,0); const startOfTodayMs = d.getTime();`. Query chat_messages using the `by_createdAt` index with `.filter((q) => q.gte(q.field("createdAt"), startOfTodayMs))` and filter further by `q.eq(q.field("tenantId"), args.tenantId)` — since chat_messages has a compound index, use `.withIndex("by_tenantId_waUserId", q => q.eq("tenantId", args.tenantId))` to scope to tenant and then `.collect()` all tenant messages, then count those where `createdAt >= startOfTodayMs` in JS. (Convex range queries on non-leading index fields need JS filtering.)
- `bookingsThisWeek`: count of booking_drafts with `status === "confirmed"` and `confirmedAt >= startOfWeekMs`. Compute start-of-week: `const dow = d.getDay(); const diff = d.getDate() - dow; const weekStart = new Date(d.setDate(diff)); weekStart.setHours(0,0,0,0); const startOfWeekMs = weekStart.getTime();`. Use `.withIndex("by_tenantId_waUserId", q => q.eq("tenantId", args.tenantId))`.collect() and filter in JS.
- `botStatus`: return `tenant.status` ("active" | "disabled"). Fetch tenant with `ctx.db.get(args.tenantId)`.
- Add `botStatus: tenant?.status ?? "active"` to the return shape.
- Keep all existing fields — do not remove anything.
